Timer example, code is here

I think this is a very nice solution, you can create instances of the timer function and pass the container where the timer should be created. Also, it could be cached, I planned to do so, but it was not in the requirements, you could modify the code minimally and add some caching so, the time count refers the cached initial time instead of just regular time. For the template, I know it looks a bit bad to have it in a string literal, but, considering the size of this problem, no constraints were given for such a small project, I am aware I could keep the template in a separate file and load as many templates needed using something like require.js or pure js. There were no specifications to add months, years, etc...
